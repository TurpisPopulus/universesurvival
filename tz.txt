Это информация по проекту, ориентируйся на нее.
Папка server содержит файлы сервера
Сервер написан на языке C#
Файл docs_server должен содержать подробную инструкцию по серверу
Файл ver_server должен содержать версию сервера в формате v0.000 / дата / список изменений.
Каждый раз изменяя файлы сервера описываешь ключивые изменения в файле ver_server добавляешь уже к существующим записям увиличивая версию на 0.001

Папка universesurvival содержит файлы клиента
Клиент использует Godot
Скрипты на языке Godot
файл docs_client должен содержать подробную инструкцию по клиенту
Файл ver_client олжен содержать версию клиента в формате v0.000 / дата / список изменений 
Каждый раз изменяя файлы клиента описываешь ключивые изменения в файле ver_client добавляешь уже к существующим записям увиличивая версию на 0.001
